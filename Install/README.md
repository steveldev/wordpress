# Installation

## Manual installation
https://fr.wordpress.org/

## Install with composer
See https://wpackagist.org/

- Create app directory
- In app directory
- Copy composer.json
- Run composer install
- Add Symilinks

## Install with SSH
```
mkdir wordpress
cd wordpress
wget https://wordpress.org/latest.tar.gz
tar xzvf latest.tar.gz
rm latest.tar.gz
mv wordpress wp-dev
```
